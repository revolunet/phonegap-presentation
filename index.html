What is Phonegap ?





An application container for your web app

Provides an cross-platform API to native functions

Allows single code base for various platforms

iOS, Android, BlackBerry, Win7/8, Tizen, Symbian...

Allows application distribution through stores

What is Phonegap ?





Phonegap native API anticipates browsers evolutions

And strictly follows the W3C recommendations

Does not provide the UI layer

Works with any HTML/Javascript framework

HTML5 progress









HTML5 features adoption (Sencha, 12/12)

Phonegap History





Started as a 24hr hackaton project in 2009

Developed during two years for various projects @Nitobi

Team acquired by Adobe in Sept, 2011

Now a key feature in Adobe mobile strategy

And an Open Source Apache project (Cordova)
